# @motionforge/showcase

Private workspace package containing the scenes used by the playground and open-source showcase docs.

The package exists so demos are authored once, validated in tests, and reused by examples instead of drifting across copied JSON files.

## API Stability

`@motionforge/showcase` is private and not part of the public API surface. Import it only from workspace apps, examples, and tests. Public users should copy generated scene JSON from `examples/generated` or use the documented authoring/preset packages instead.
